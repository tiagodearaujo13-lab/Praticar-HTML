# Praticando HTML5 Puro: Projeto Universitário Fictício

Olá! Este repositório é dedicado à minha jornada de aprendizado e **prática intensiva de HTML5**.

O principal objetivo deste projeto é dominar a **estrutura semântica do HTML**, aprofundando o uso correto de tags e garantindo a acessibilidade.

## Objetivo Final do Projeto

Ao final deste ciclo de prática, o projeto se transformará em:

**Uma Página Estática Completa para uma Universidade Fictícia.**

O foco está na organização e na clareza do código HTML, utilizando a estrutura mais moderna e limpa possível.

### Tecnologias Focadas

* **HTML5 Puro:** Sem frameworks ou bibliotecas, apenas a fundação.
* **Semântica Web:** Uso correto de tags.

Fique à vontade para acompanhar a evolução do código!
